Variable replacement
----------------
2020-09-24


- Light_RealGenerator: !{var}, pool: variables
- Light_Nugget: %{var}, pool: _vars
- Container: ${var}, pool: the config file
- Light_Kit: ${var}, pool: via the AdminPageController->renderAdminPage method (lka)
